# Sistema de Facturación Completo - DeCamino

## Módulo de Facturación
- Emisión de facturas profesionales
- Generación automática de PDF
- Gestión de estados y seguimiento
- Integración con sistema eFactura

## Módulo de Gastos
- Carga de facturas de proveedores
- Procesamiento automático con OCR/AI
- Extracción de datos automática
- Conciliación con facturación

## Módulo de Catálogo
- Gestión de productos y servicios
- Organización por categorías
- Control de stock y precios
- Exportación a Excel

## Módulo de Facturas Recibidas
- Gestión de facturas de proveedores
- Control de estados y pagos
- Seguimiento de gastos
- Exportación a Excel

## Módulo de Impuestos
- Gestión de IVA (303, 390)
- Control de IRPF (130, 100)
- Retenciones y obligaciones
- Calendario fiscal

## Descripción
Sistema completo de gestión financiera integrado en la aplicación DeCamino para servicios auxiliares.
